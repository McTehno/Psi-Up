# Zaslonske maske

Mapa vsebuje izbrane zaslonske maske aplikacije **NIDiKo**.

Slike prikazujejo ključne dele uporabniške izkušnje: začetni pregled aplikacije, raziskovanje učnih vsebin, vprašalnik in AI pomoč.

## Predvidene slike

- `01-domaca-stran.png` — začetni prikaz aplikacije.
- `02-iskanje.png` — iskanje po učnih poteh, modulih in učnih enotah.
- `03-ucne-poti.png` — pregled učnih poti.
- `04-detail-stran.png` — podrobnosti izbrane učne vsebine.
- `05-vprasalnik.png` — vprašalnik za oceno trenutnega znanja.
- `06-ai-pomoc.png` — pogovor z AI pomočnikom pri razumevanju vsebin ali vprašanj.

Zaslonske maske so izbrane tako, da prikažejo glavni tok aplikacije: od raziskovanja vsebin do vprašalnika in podpore pri razumevanju.
